Zookeeper Project

How to run in PyCharm:
1. Open the zookeeper_project folder.
2. Right-click main.py and choose Run 'main'.
3. The program will create zooPopulation.txt in the same folder.

Files included:
- main.py
- animal.py
- hyena.py
- lion.py
- tiger.py
- bear.py
- arrivingAnimals.txt
- animalNames.txt
- zooPopulation.txt (generated sample output)
